Acest proiect reprezintă o soluție software integrată, dezvoltată în colaborare de Hara Andrada Iarina și Iancu Cezara Andreea.
Arhitectura sistemului reunește două componente principale: o platformă Web construită pe tehnologia .NET Razor Pages și o aplicație mobilă nativă dezvoltată în .NET MAUI.
Cele două module sunt interconectate, asigurând o comunicare fluidă și sincronizarea datelor între mediul web și dispozitivele mobile
